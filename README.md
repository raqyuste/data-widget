## Architecture
 - `views`: the visual UI
 - `middlewares`: application logic
 - `store`: the application's models state
 - `service`: the connection with the api

## Installing
```sh
git clone git@github.com:raqyuste/data-widget.git
cd data-widget && npm install
```

## Local serve
```sh
npm start
```

## Building
```sh
npm run build
```

## Testing
```sh
npm test
```
